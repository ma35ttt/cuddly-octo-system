# Dr. Sharma's Smile Care Dental Clinic — Website

A modern, fully responsive dental clinic website built with HTML, CSS, and JavaScript.

## 🦷 Live Demo
Host on GitHub Pages by following the steps below.

## 📁 Project Structure

```
smile-care-dental-clinic/
├── index.html          ← Main webpage
├── css/
│   └── style.css       ← All styles & responsive design
├── js/
│   └── main.js         ← Scroll animations, form, mobile menu
├── assets/             ← Place any images/icons here
└── README.md
```

## 🚀 Deploy to GitHub Pages (Free Hosting)

1. Create a new repository on [github.com](https://github.com)
2. Name it: `smile-care-dental-clinic` (or any name)
3. Upload all files keeping the folder structure intact
4. Go to **Settings → Pages**
5. Under **Source**, select `main` branch → `/ (root)`
6. Click **Save** — your site will be live at:
   `https://yourusername.github.io/smile-care-dental-clinic`

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll-reveal animations
- Fixed navbar with scroll effect
- Hero section with clinic info card
- 8 service cards with hover effects
- About section with doctor profiles
- 6 patient testimonials

- Google Maps embed

- HTML5 (semantic markup)
- CSS3 (CSS variables, Grid, Flexbox, animations)
- Vanilla JavaScript (no frameworks, no dependencies)
- Google Fonts: Cormorant Garamond + DM Sans
